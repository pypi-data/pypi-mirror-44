<Patelier>
This "patelier" is a Python package
    for showing text lines in a text file.
The patelier comprises pateliercoredmy.py.
The pateliercoredmy.py is a test module for show text lines.
The text lines are read from input text file.
The text file needs to be encoded in UTF-8 or Shift-JIS.
This changes each control code (x00 - x1f) into 'â–²'.
